# nodejs-sandbox
typescript, node, postgres, prisma

## system reqs
make sure you have the following installed:
- docker
- node v18
- yarn v1
- vscode
- postgres

## spinning up your env
make sure you create a new `.env` file (just copy over whats in `.env.example`)
```
yarn rebuild:be
```

## tearing down your env
```
docker compose down
```
